Module information:

Access Control and Attendance App for Odoo users

The RFID Attendance app gives an upgrade to the usual HR system that your company is using. Suitable for Odoo versions 10.0 and 11.0, the TCP/IP Interface Module gives you much more opportunities. It is a required support for the Polimex Holding’s controllers and LAN modules. All our controllers use our http based WebSDK protocol.

Quick integration and access

The RFID Attendance App is very quick to install and very easy to use. It also contains an open source code so it is very easy to integrate it to already useful system. You are just a few clicks away from the whole-decision for your business!

More information at one system

The RFID Attendance App guarantees you the possibility to control all the information about the entrance in a specific building or place. Once created, a list with all employees data is usable to gain information about their access by authorization- in buildings/rooms, teams or proper work, done at the moment. The system also informs you when one of your colleagues is sick, is on a paid leave or did not came to work. That information flows directly to the managers of the company, so they can make their decisions.

Optional - the RFID Attendance App can give you control over the vending machine at the office. You just have to ask for it.

Real time metrics

The best thing is all this happens in real time! All the data is accessible from any platform, anywhere . You receive information even for different objects you manage. It ups to you whether you are going to use the app by mobile phone, tablet or PC. So, anytime – anywhere is a good decision for those who want to know everything about the business.

Module tutorials

Module installation instructions:

RFID Access Control allows you to communicate with Polimex TCP/IP Interface Modules and adds some features to the HR app. The RFID Attendance module integrates this one with the HR Attendance module.

If you have a different attendance module or you prefer to create your own module, you can check out the source code for RFID Attendance: https://github.com/polimex/odoo_apps. It's pretty short and simple.

  1. Install the RFID Access Control app:

  2. Now you need to give yourself access to the RFID menus and controls.
    Warning: The menus might look very differently for your Odoo installation.
    Note: All administrators can change these settings.

    1. Go to Odoo Settings:

    2. Go to Users:

    3. Click on the user you want to give access to

    4. Click on edit:

    5. In the "RFID Access" menu, set the permissions accordingly:

      • Officer allows the user access to the User and the System events only.
      • Manager allows the user access to all the menus.

  3. Now you need to go to your Polimex TCP/IP Interface Module and set the SDK settings accordingly:

    1. SDK sends events: Enable
    2. RPC JSON format (Odoo): Enable
    3. HTTP Server Push URL: *your.odoo.server*/hr/rfid/event
    • Note: Please change *your.odoo.server* with a link or an ip to the odoo server
    • Note: Polimex TCP/IP Interface Module currently work through HTTP and not HTTPS.

How to activate your Polimex TCP/IP Interface Module:

  • Warning: You must have Manager access to the RFID Access Control app
  • Warning: You must have a controller connected to your Polimex TCP/IP Interface Module
  1. Check out step 3 of the installation instructions to link it with Odoo if you haven't already
  2. After scanning for controllers on your Polimex TCP/IP Interface Module, create an event on the controller (touch your card/tag to one of the readers on some controller)
  3. The Polimex TCP/IP Interface Module will send the event to Odoo, it should appear on the Modules page:
  4. Click on the newly created module (you should recognize it using the Serial Number) and click on the "Set module to active" button. If the button is missing then the module is already active:
  5. Your module is now active and Odoo will accept events from it.

How to add a card to Odoo:

  • Warning: You must have Manager access to the RFID Access Control app
  • Warning: Your Polimex TCP/IP Interface Module must already be linked with Odoo
  • Warning: Your Polimex TCP/IP Interface Module must be active
  • Warning: You must have a controller that can read cards connected with your Polimex TCP/IP Interface Module
  1. Touch your Tag/Card with the controller connected to your Polimex TCP/IP Interface Module. The device will now send an event to Odoo.
  2. Under System Events an event should appear on the top of type "Could not find a card with this number":
  3. Copy the card number from the json object in the description:
  4. Go to cards and click on "create":
  5. Paste the card number under "Card Number", choose a card owner (a person from the HR module) and a card type:
  6. After pressing the save button, your card will be created.
